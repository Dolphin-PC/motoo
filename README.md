## Intro Project

### '우리 모두 투자하자'

> 모투서비스는 모의투자 경험을 통해, 증권시장에 대한 이해를 돕기 위한 서비스입니다.

- 한국투자증권의 모의투자 API를 연동한 웹서비스입니다. (추후 모바일앱으로 제작 예정(RN))
- 한국투자증권에서 발급받은 `appkey`와 `appsecret`을 서비스에 등록하여, 사용자의 모의계좌를 통해 모의투자를 시작합니다.
- 배포 : https://motoo-a.vercel.app/
